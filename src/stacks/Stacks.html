Stack


A stack is an ordered list in which insertion and deletion are done at one end, called top. Stack follows Last In First Out paradigm.


Stack Operations

  • push(int data): inserts data onto stack
  • int pop(): removes and returns the last inserted data from the stack
  • int peek(): returns the last inserted element without removing it
  • int size(): returns the count of elements in the stack
  • boolean isEmpty(): checks if stack is empty
  • boolean isFull(): checks if stack is full

    • When an element is popped or peeked from an empty stack, EmptyStackException is thrown.

    Space Complexity: O(n) to hold n elements

    Time Complexity: O(1) for pop, peek, size, isEmpty, isFull, O(1) for push using linkedlist, O(n) for push using dynamic array


    1. ImplementStackUsingArrays

    Using Static Arrays vs Dynamic Arrays for implementation: When used static arrays, maximum size of stack must be defined in prior and cannot be changed. Item cannot be pushed into a full stack.

    When used dynamic arrays, for every push, array size is incremented. Instead, when used repeated doubling technique, every time array reaches its max limit, array size is doubled to the existing size. Amortized time-complexity to push n elements will be O(n)

    2. ImplementStackUsingLinkedList

    Push operation is implemented by inserting node at the beginning of the list (head) Pop operation is implemented by deleting node from the beginning of the list (head)
